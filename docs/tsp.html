dist_route(route: list[int]) → float[source]

Compute the total distance the traveller should travel when executing the route. If the route is of the form \(x = (x_0, \ldots, x_n)\), then the total distance is computed by

\[\sum_{k=0}^{n} d(x_k, x_{k+1})\]

With the convention that \(x_{n+1} = x_0\).

Parameters:

route – The route to compute the total distance.

Returns:

The total distance.

tsp.utils.logarithmic_cooling_schedule(k, T_0=25, k_0=1) → float[source]

The logarithmic cooling schedule is

\[T_k = T_0 \cdot \ln(k_0 + 1) / \ln(k + 1 + k_0)\]
Parameters:

  • k (int) – The number of the iteration at the temperature \(T_k\).

  • T_0 (float) – Initial temperature. Default is 25

  • k_0 (float) – Iteration phase shift. Default is 1

Returns:

The temperature at iteration \(k\).
